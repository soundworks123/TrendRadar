# 分析要求

对每个热点，请提供以下维度的分析：

## 1. 爆款价值评分（1-10分）

**评分标准：**
- **9-10分（超级爆款）**：全民热议，跨圈层传播，必出爆款
- **7-8分（高爆款）**：广泛关注，易出10万+阅读
- **5-6分（中等爆款）**：圈层热点，稳定流量
- **3-4分（低爆款）**：小众话题，垂直领域
- **1-2分（无爆款）**：无传播价值

## 2. 爆款角度分析

分析以下要点：
- 公众最关心的核心点是什么？
- 争议点/共鸣点在哪里？
- 情绪触发点（愤怒/焦虑/好奇/感动）
- 如何切入最容易引爆？

## 3. 推荐内容形式

分析适合的内容形式：
- 深度长文 / 短视频 / 图文 / 漫画 / 播客
- 推荐发布平台（公众号/抖音/B站/小红书/知乎）
- 内容时长/字数建议

## 4. 目标受众画像

- 年龄段
- 职业特征
- 兴趣标签
- 受众规模估算

## 5. 创作建议

提供具体可执行的建议：
- **标题风格**：震惊体 / 干货型 / 情感型 / 故事型 / 数据型
- **标题示例**：给出3-5个具体标题
- **内容结构**：开头 / 中间 / 结尾如何安排
- **关键素材**：需要哪些数据、案例、金句
- **发布时机**：最佳发布时间
